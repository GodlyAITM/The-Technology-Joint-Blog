# The Technology Joint

## Vision

The Technology Joint is a founder-led technology publication focused on practical technology education, software, cybersecurity, AI, smartphones, and industry news.

## Mission

To help readers make better technology decisions through trustworthy journalism, expert reviews, and actionable guides.

## Tech Stack

- Astro 7
- TypeScript
- Markdown Content Collections
- GitHub
- GitHub Pages (initial deployment)

## Principles

- Performance First
- SEO First
- Accessibility by Default
- Reusable Components
- Content Driven
- Mobile First
